# Email Sender Component

Sends emails via SMTP or third-party email APIs. Used for delivering generated content, reports, or notifications to recipients.

## Overview

The Email Sender component sends formatted emails with support for HTML content, attachments, CC/BCC recipients, and multiple email provider backends including SMTP, SendGrid, and AWS SES.

## Inputs

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `to` | `string` or `list[string]` | Yes | Recipient email address(es) |
| `subject` | `string` | Yes | Email subject line |
| `body` | `string` | Yes | Email body (plain text or HTML) |
| `from_email` | `string` | Yes | Sender email address |
| `provider` | `dropdown` | No | Email provider: `smtp`, `sendgrid`, `ses` (default: `smtp`) |
| `is_html` | `bool` | No | Whether the body contains HTML (default: `false`) |
| `cc` | `list[string]` | No | CC recipients |
| `bcc` | `list[string]` | No | BCC recipients |
| `attachments` | `list[file]` | No | Files to attach |
| `smtp_host` | `string` | No | SMTP server host (for SMTP provider) |
| `smtp_port` | `int` | No | SMTP server port (default: 587) |
| `api_key` | `secret` | No | API key (for SendGrid/SES providers) |

## Outputs

| Name | Type | Description |
|------|------|-------------|
| `message_id` | `string` | Unique message identifier |
| `is_success` | `bool` | Whether the email was sent successfully |
| `error` | `string` | Error message if sending failed |

## Usage Example

```python
from custom_components import EmailSenderComponent

email = EmailSenderComponent()
result = email.run(
    to="team@example.com",
    subject="Weekly AI Report",
    body="<h1>Summary</h1><p>Processed 500 queries this week...</p>",
    from_email="ai-pipeline@example.com",
    provider="sendgrid",
    api_key="SG.your-key",
    is_html=True
)
```

## Common Use Cases

- Sending AI-generated reports to stakeholders
- Delivering summarized documents to subscribers
- Email notifications for pipeline completion or failure
- Distributing personalized content generated by LLMs
